# Project Overview

Project Name

AI Habit Platform

Goal

Build a backend portfolio project demonstrating:

- NestJS backend API
- Python AI service
- OCR text extraction
- LLM data structuring
- PostgreSQL + MongoDB
- External API integration (Google Calendar, FCM)
- Log pipeline with ELK Stack (Elasticsearch, Logstash, Kibana, Filebeat)

Example user scenario

1 User uploads food label or receipt image
2 OCR extracts text from the image
3 LLM converts the text into structured nutritional data
4 Backend stores the structured data
5 Reminder events are created in Google Calendar
6 Push notifications remind the user via FCM
7 All service logs are collected and visualized via ELK Stack

Purpose

Demonstrate real-world backend architecture integrating AI services, external APIs, and log monitoring infrastructure.
