# Project Overview

Project Name

AI Habit Platform

Goal

Build a backend portfolio project demonstrating:

- NestJS backend API
- Python AI service
- OCR text extraction
- LLM data structuring
- PostgreSQL + MongoDB
- External API integration

Example user scenario

1 User uploads food label or receipt image  
2 OCR extracts text from the image  
3 LLM converts the text into structured nutritional data  
4 Backend stores the structured data  
5 Reminder events are created  
6 Push notifications remind the user

Purpose

Demonstrate real-world backend architecture integrating AI services.
